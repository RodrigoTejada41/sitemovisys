# 17/07/2026 — Seção Nossos Clientes

## Criado

- Seção institucional `Nossos Clientes` integrada após os segmentos atendidos.
- Carrossel automático desktop e grid responsivo mobile.
- Componentes reutilizáveis `ClientsSection` e `ClientLogoCard`.

- Estilos TailwindCSS isolados do CSS existente.
