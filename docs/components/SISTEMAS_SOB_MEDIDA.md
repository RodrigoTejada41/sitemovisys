# Sistemas sob medida

Implementado em 17/07/2026 na seção `Experiência aplicada`.

## Objetivo

Apresentar o desenvolvimento de sistemas personalizados antes da lista de segmentos atendidos.

## Comportamento

- Bloco institucional informativo, sem link ou botão.
- Lista recursos de sistemas web e desktop, integrações, automações, evolução e suporte.
- Os segmentos permanecem abaixo e continuam interativos.
- Layout responsivo em desktop, tablet e celular.
